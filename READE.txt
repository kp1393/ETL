***********************************Projeto Cidades******************************


O objetivo dessa aplicação é demonstrar a utilização da linguagem Python para a busca de informações de cidades brasileiras no site do IBGE por meio da biblioteca requests.

A partir do Código IBGE, que é um identificador para cada município do Brasil, é realizada a busca pelos dados no site e, após o processamento pelo Python, os dados são exportados para uma planilha online (Google Sheets) onde será possível visualizar o trabalho. 

O arquivo projeto-napp-f881b2552721.json é usado para auxiliar na conexão com a planilha
